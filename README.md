# NOUN Study Buddy 🎓

A robust, Django-based academic resource management and CBT (Computer-Based Test) practice platform tailored for Computer Science students at the National Open University of Nigeria (NOUN).

## 📖 About the Project

University students often struggle with fragmented academic resources and lack of centralized platforms for exam preparation. **NOUN Study Buddy** solves this by providing a unified, scalable backend architecture where students can access course materials, download textbooks, and track their curriculum.

Currently under active development, the platform is designed to handle complex relational data between courses, academic departments, and structured exam practice questions.

## 🚀 Key Features

* **Course Management:** Relational database structure to manage course codes, descriptions, and premium/free tiers.
* **Resource Sharing:** Secure file handling for PDF textbook uploads and downloads.
* **CBT Practice Engine:** Dynamic question modeling linked to specific courses for targeted exam preparation.
* **Scalable Architecture:** Built with a modular Django app structure (`courses`, `users`, etc.) for easy future expansion.

## 🛠️ Tech Stack

* **Backend Framework:** Python / Django
* **Database:** SQLite (Development) / PostgreSQL (Production ready)
* **Architecture:** MVT (Model-View-Template) / RESTful API design principles
* **Version Control:** Git & GitHub
